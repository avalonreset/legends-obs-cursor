# Code Of Conduct

This project is maintained for practical, respectful collaboration around an OBS
creator tool.

## Expected Behavior

- Be direct and constructive.
- Keep feedback focused on the project.
- Respect the original art direction and contributor work.
- Avoid posting secrets, private OBS configs, or personal workstation details.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private configuration, credentials, or recordings.
- Derailing issues with unrelated promotion or spam.

## Enforcement

Maintainers may edit, close, lock, or remove issues and pull requests that do
not follow this code of conduct.
