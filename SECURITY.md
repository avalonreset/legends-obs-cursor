# Security Policy

## Supported Versions

The current `main` branch is the supported development line until the first
tagged release.

## Reporting A Vulnerability

Please report security issues privately through GitHub Security Advisories once
the repository is public. While the repository is private, contact the maintainer
through the Avalon Reset GitHub account.

## Scope

This project is an OBS Lua script and PowerShell installer. Relevant issues
include unsafe file writes, unsafe scene mutation, untrusted download behavior,
or accidental exposure of local OBS configuration.

Do not include secrets, OBS profiles, scene collections, recordings, or local
machine paths in public issues.
